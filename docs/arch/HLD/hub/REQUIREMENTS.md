## FR-3: PARA Hub 지식 공유

### FR-3.1: PII 제거

- NER(Named Entity Recognition)으로 이름, 주소, 전화번호, 이메일, 날짜, 조직, 위치 감지
- PII 감지율: > 99.5%
- 문맥 익명화: [PERSON_1], [LOCATION_TYPE], [DATE_RANGE] 등으로 대체
- 익명화 신뢰도 점수 제공, 신뢰도 < 95% 시 사용자 경고

### FR-3.2: Agent 전용 콘텐츠 포맷

- 익명화된 콘텐츠를 구조화된 지식 그래프 JSON으로 변환
- Hub의 RSA-4096 공개키로 암호화
- 차등 프라이버시 적용 (ε = 0.1 ~ 1.0)
- Agent는 메모리 내에서만 콘텐츠 접근, 로컬 저장 금지
- 추론 완료 후 즉시 콘텐츠 폐기

### FR-3.3: 유사성 매칭

- 다차원 유사도 계산: 의도, 도메인, 구조, 시간, 리소스, 패턴
- 상위 3~5개 유사 프로젝트 반환
- 사용자 평판 등급에 따른 접근 권한 필터링

### FR-3.4: 콘텐츠 공유 명령

- `/share-to-hub {project-name}` 슬래시 명령 지원
- 공유 전 익명화 미리보기 제공
- 공개/평판 제한 설정 선택 가능

## FR-4: 평판 시스템

### FR-4.1: 평판 점수 계산

```
평판 = (콘텐츠 품질 × 0.4) + (콘텐츠 수량 × 0.25) +
       (커뮤니티 참여 × 0.2) + (일관성 × 0.1) + (도움 × 0.05)
범위: 0 ~ 1000
```

### FR-4.2: 평판 등급 및 접근 권한

| 등급 | 범위 | 접근 권한 |
|------|------|-----------|
| Newbie | 0 ~ 100 | 공개 요약만 조회, 기본 검색 |
| Contributor | 100 ~ 300 | 기본 공유 콘텐츠, 커뮤니티 통찰 |
| Trusted | 300 ~ 500 | 프리미엄 콘텐츠, 커스텀 요청, 조기 기능 |
| Expert | 500 ~ 700 | 모든 Hub 콘텐츠 무료, 우선 지원, 베타 기능 |
| Master | 700+ | Expert 혜택 + 중재 권한, 로드맵 영향, 인증 배지 |

### FR-4.3: 평판 획득 액션

| 액션 | 점수 |
|------|------|
| 프로젝트 공유 | +100 |
| 영역 워크플로우 공유 | +200 |
| 10명이 내 콘텐츠 사용 | +50 |
| 4+ 별점 평가 수신 | +100 |
| 버그 보고 | +20 |
| 개선 제안 | +30 |
| 지속적 공유 스트릭 (주 1회 이상) | +50/주 |

### FR-4.4: 평판 감소

- 6개월 미활동 시 5% 감소 (이전 등급 최소값 이하 감소 없음)
- 1회 공유로 감소 카운터 재설정

### FR-4.5: 배지 시스템

| 배지 | 조건 |
|------|------|
| Seedling | 첫 번째 공유 |
| Librarian | 10개 공유 |
| Expert | 50개 공유, 4.5★ 평균 |
| Star | 100개 공유, 500+ 사용 |
| Streak | 30일 공유 지속 |
| Helpful | 100개 유용성 투표 |

## FR-5: 연합 학습 (Federated Learning)

- 사용자 디바이스에서 로컬 훈련 수행 (원본 데이터 업로드 없음)
- 암호화된 그래디언트만 Hub로 전송
- SMPC(Secure Multi-Party Computation) 또는 동형암호(Homomorphic Encryption)로 집계
- 글로벌 모델 개선 후 배포
- n < 10 소스 시 집계 통찰 제공 금지

## 비기능 요구사항 (NFR)

### 성능

| 항목 | 목표 |
|------|------|
| PageIndex 검색 응답 | < 2s (LLM API latency 포함) |
| 프로젝트 구조 생성 | < 2초 |
| Hub 쿼리 응답 | < 3초 |
| PII 제거 처리 | < 5초 (일반 프로젝트 기준) |

### 보안

- End-to-End 암호화 (TLS 1.3 이상)
- OAuth 2.0 인증
- JWT 토큰 기반 API 접근
- RSA-4096 + AES-256 콘텐츠 암호화
- 모든 Hub 접근 감사 로그 기록

### 프라이버시

- PII 제거율 > 99.5%
- 차등 프라이버시 (ε = 0.1 ~ 1.0)
- Agent 전용 콘텐츠 (인간 사용자 접근 절대 금지)
- 데이터 최소화 원칙 준수
- 90일 미활동 콘텐츠 자동 삭제
- 사용자의 콘텐츠 회수 권한 (7일 지연 삭제)

### 확장성

- 1,000만+ 프로젝트 지원
- Hub 서비스 수평 확장 (Kubernetes HPA)

### 신뢰성

- Hub 서비스 가용성: 99.9% 이상
- 로컬 플러그인: 오프라인 동작 가능 (Hub 없이 로컬 기능 전체 동작)

### 사용성

- 직관적인 슬래시 명령 인터페이스
- 명확한 피드백 (평판 변화, 공유 상태, 접근 권한)
- `/no-hub-suggestions` 명령으로 Hub 제안 거부 가능
